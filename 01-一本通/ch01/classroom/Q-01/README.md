## 题目代码

#### C 实现版本

```c
#include <stdio.h>

int main() {
    printf("Hello World!\n");
    return 0;
}
```

#### C++实现版本

```c++
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
    return 0;
}
```
