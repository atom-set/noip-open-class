# 第一章: C++语言入门

## 本章小结

- 1.程序由 1 个或多个函数组成，必须有且只有一个主函数 main，程序从 main 函数开始执行
- 2.程序中的变量使用前必须先定义数据类型
- 3.程序调用调库函数前需包含相应头文件，比如 `<cstdio>` 和 `<iostream>`
- 4.程序中可以有注释行，以增强程序的可读性和可理解性
- 5.程序的语句以分号结束

## 课题练习 1: 计算机输出

#### 【题目描述】

在屏幕上输出“Hello World!”。

#### 输入输出要求

【输入】

(无）

【输出】

(无）

#### 测试用例

【输入】

(无）

【输出】

Hello World!

## 题目代码

```c++
#include<cstdio>
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
    return 0;
}
```
