# 第一章: C++语言入门

## 课后练习 4：## 字符三角形

#### 【题目描述】

给定一个字符，用它构造一个底边长 5 个字符，高 3 个字符的等腰字符三角形。

#### 输入输出要求

【输入】

输入只有一行，包含一个字符。

【输出】

该字符构成的等腰三角形，底边长 5 个字符，高 3 个字符。

#### 测试用例

【输入样例】

`*`

【输出样例】

```
  *
 ***
*****
```

## 题目代码

```c++
#include <cstdio>
#include <iostream>
using namespace std;

int main() {
    char ch;
    cin >> ch;
    cout << "  " << ch << endl;
    cout << " " << ch << ch << ch << endl;
    cout << ch << ch << ch << ch << ch << endl;

    return 0;
}
```
