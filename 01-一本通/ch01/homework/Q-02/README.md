# 第一章: C++语言入门

## 课后练习 2：输出第二个整数

#### 【题目描述】

输入三个整数，整数之间由一个空格分隔，整数是 32 位有符号整数。把第二个输入的整数输出。

#### 输入输出要求

【输入】

只有一行，共三个整数，整数之间由一个空格分隔。整数是 32 位有符号整数。

【输出】

只有一行，一个整数，即输入的第二个整数。

#### 测试用例

【输入样例】

123 456 789

【输出样例】

456

## 题目代码

```c++
#include <iostream>
#include <cstdio>
using namespace std;
int main()
{
  // 有符号整数的定义
  signed int a, b, c;
  cin >> a >> b >> c;

  printf("%d\n", b);
  return 0;
}
```
