# 3086：【例 16.1】 判断数正负

> 需要 5 个 case

> 时间限制 1000 ms

> 内存限制: 65536 KB

## 题目描述

给定一个整数 N，判断其正负。如果 N>0，输出 positive；如果 N=0，输出 zero；如果 N<0，输出 negative。

## 输入输出要求

【输入】

> 个整数 N `(−10^9≤N≤10^9)`。

【输出】

```
如果 N>0, 输出 positive;
如果 N=0, 输出 zero;
如果 N<0, 输出 negative。
```

## 测试用例

【输入样例】

> 1

【输出样例】

> positive
