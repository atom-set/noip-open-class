# 3092：【例 17.1】 双面打印

> 需要 5 个 case

> 时间限制 1000 ms

> 内存限制: 65536 KB

## 题目描述

为了方便小蓝的学习，妈妈会把老师发放的材料打印出来。而为了环保，妈妈一般都进行双面打印，也就是一张纸的正反面都打印出相应的内容。举个例子来说：如果一份电子材料有 3 页，那么需要 2 张纸进行打印；如果一份电子材料有 4 页，那么还是需要 2 张纸进行打印。现在已经知道了一份电子版的学习材料的页数 N ，你能帮小蓝计算一下需要几张纸吗？

## 输入输出要求

【输入】

> 整数 N `(0≤N≤1000 )`，代表一份电子版的学习材料的页数。

【输出】

> 双面打印所需纸的张数。

## 测试用例

【输入样例】

> 7

【输出样例】

> 4
