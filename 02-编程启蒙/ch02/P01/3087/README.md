# 3087：【例 16.2】 开灯

> 需要 5 个 case

> 时间限制 1000 ms

> 内存限制: 65536 KB

## 题目描述

小蓝家的灯是线型开关的，拉一次灯开，再拉一次灯关，未拉之前是熄灭状态。

输入一个正整数 M `(1<M<100 )`，作为小蓝拉灯的次数，判断拉灯 M 次后，灯是点亮状态还是熄灭状态。

## 输入输出要求

【输入】

> 输入一个正整数 M 作为拉灯的次数 `（1<M<100）`。

【输出】

> 如果灯是点亮状态输出整数 1 ，如果灯是熄灭状态输出整数 0 。

## 测试用例

【输入样例】

> 5

【输出样例】

> 1
