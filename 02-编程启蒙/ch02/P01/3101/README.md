# 3101：练18.1 还要等多久

> 需要 5 个 case

> 时间限制 1000 ms

> 内存限制: 65536 KB

## 题目描述

晶晶又想打电话给贝贝，可是这次他碰到了另一个问题。晶晶不想打扰贝贝课，也不想才刚打通电话她就要上课去了，因此他决定一定要在刚好 25 分钟的时候打电话给她。给你现在的时间的分钟，请你帮他算算看还要等多久才能打电话给贝贝？

## 输入输出要求

【输入】

> 输入只有一行，包含现在时间的分钟 m `(0≤m≤59 )`。

【输出】

> 输出还要等几分钟晶晶才能打电话。

## 测试用例

【输入样例】

> 20

【输出样例】

> 5
