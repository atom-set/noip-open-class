# 3098：【例 18.1】 睡眠

> 需要 5 个 case

> 时间限制 1000 ms

> 内存限制: 65536 KB

## 题目描述

一个人只有每天睡眠时间到达 8 小时才算是睡够的。现在统计出某个人的中午睡觉时间 x 分钟，晚上睡觉时间 y 小时，请你计算他是否睡眠充足。

已知，中午睡觉 1 分钟相当于晚上睡觉 3 分钟， 8 小时是以晚上睡觉时间为标准。

## 输入输出要求

【输入】

> 两个整数 `x (0≤x≤1000 ),y (0≤y≤7 )`，分别表示中午睡觉的分钟数和晚上睡觉的小时数。

【输出】

> 如果睡眠时间充足，输出 `Yes`；否则输出 `No`。

## 测试用例

【输入样例】

> 40 6

【输出样例】

> Yes
