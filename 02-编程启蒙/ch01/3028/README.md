# 3028：【例 7.2】与圆相关的计算

> 需要 5 个 case

> 时间限制: 1000 ms

> 内存限制: 65536 KB

## 题目描述

给出圆的半径，求圆的直径、周长和面积。

如果圆的半径是 r，那么圆的直径、周长、面积分别是 `2×r`、`2×π×r`、`π×r×r`，其中约定 π=3.14159。

## 输入输出要求

【输入】

> 输入包含一个实数 r `(0<r≤10,000)`，表示圆的半径。

【输出】

> 输出一行，包含三个数，分别表示圆的直径、周长、面积，数与数之间以一个空格分开，每个数保留 4 位小数。

## 测试用例

【输入样例】

> 3.0

【输出样例】

> 6.0000 18.8495 28.2743
