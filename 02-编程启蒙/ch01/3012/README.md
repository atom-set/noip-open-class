# 3012：练 4.1 阿布拉 1

> 时间限制: 1000 ms

> 内存限制: 65536 KB

## 题目描述

尼克是一位好学的学生，他经常做实验。有一天他在做实验的时候需要交换一瓶 10 毫升的红墨水和一瓶 20 毫升的蓝墨水，但他不会，便向狐狸老师求助，狐狸老师念了一句咒语“阿布拉”帮助了尼克。试编一程序，模拟这一过程。

## 输入输出要求

【输入】

> 无

【输出】

```
第一行输出交换前两个瓶子的墨水数量。
第二行交换后两个瓶子的墨水数量（单位ml）。
```
