# 3070：练13.2 对齐输出

> 需要 5 个 case

> 时间限制 1000 ms

> 内存限制: 65536 KB

## 题目描述

读入三个整数，按每个整数占8个字符的宽度，右对齐输出它们，按照格式要求依次输出三个整数，之间以一个空格分开。

## 输入输出要求

【输入】

> 只有一行，包含三个整数，整数之间以一个空格分开

【输出】

> 只有一行，按照格式要求依次输出三个整数，之间以一个空格分开

## 来源

参考--小学生c++趣味编程

## 测试用例

【输入样例】

> `123456789 0 -1`

【输出样例】

> `123456789       0      -1`
