# 3033：【例 8.1】人民币支付

> 需要 5 case

> 时间限制 1000 ms

> 内存限制: 65536 KB

## 题目描述

给定一个金额（以元为单位，如 345），现在想知道支付该金额的各种面额的人民币数量，显示 100 元，50 元，20 元，10 元，5 元，1 元各多少张，要求尽量使用大面额的钞票。

## 输入输出要求

【输入】

> 一个小于 1000 的正整数。

【输出】

> 输出分行，每行显示一个整数，从上到下分别表示 100 元，50 元，20 元，10 元，5 元，1 元人民币的张数。

## 测试用例

【输入样例】

> 735

【输出样例】

```
7
0
1
1
1
0
```
